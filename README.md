# Employee Salary Prediction AI

This project implements a supervised Machine Learning model using a Decision Tree Classifier to predict employee salary categories (<=50K or >50K).

## 📊 Model Performance
- Accuracy: 81.7%
- Algorithm: Decision Tree Classifier

## Features Used
- Age
- Education
- Workclass
- Occupation
- Hours per week

##  Technologies
- Python
- Pandas
- Scikit-learn

##  How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run the model:
   python salary_model.py

##  Conclusion
The model successfully predicts salary categories with good accuracy and demonstrates how machine learning can support decision-making in human resource management.
